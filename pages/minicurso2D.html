Project

Unity 2D (Mini-Course)

Platform:

Windows (Computer)

Description:

was a mini-course taught by me for free to university students, teachers and the community.
Theoretical Content:

  • Overview of Unity screens;
  • Move/View Options in Unity;
  • cameras;
  • Lighting;
  • Components;
  • Asset Store.

Practical Content:
  • 2D Vision;
  • Sprites;
  • 2D Scenery;
  • tilemap;
  • Tile Palette;
  • Sprite renderer and Tilemap Renderer;
  • 2D Collider;
  • 2D rigidbody;
  • Animation;
  • Scripts;
  • Score and character lives;
  • Buttons;
  • SavePoint;
  • Enemies;
  • Custom input commands;
  • Canvas;
  • Pause menu;
  • Game Over Screen;
  • Login screen;
  • Using MAMP to emulate a local database;
  • Basic SQL;
  • PHP and mysql;
  • Registration and login screens;
  • screens and database Interactions using Unity, C# and PHP.

Control (s):

Mouse and keyboard.
